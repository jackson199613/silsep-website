# Silsep Visual Brand & UI Style Guide

This document defines the visual soul of Silsep's digital ecosystem. All agents must adhere to these standards when generating content, UI components, or marketing materials.

## 1. Core Visual Philosophy
*   **Precision over Clutter**: Every element must serve a purpose. Alignment is non-negotiable.
*   **Intentional Technology**: We use tech-forward aesthetics (neon, glassmorphism) not as a trend, but to represent the clarity of our molecular solutions.
*   **The "Human-Agent" Synergy**: Visuals should feel futuristic yet approachable (Q-bot aesthetics).

## 3. Typography
*   **Primary (UI/Prose)**: `Inter` - Clean, professional, high readability.
*   **Technical (Data/Code)**: `Roboto Mono` - For technical specs and merit tracking.

## 4. UI Patterns (The Dashboard)
*   **Glassmorphism**: Use `backdrop-filter: blur(12px)` and subtle semi-transparent borders.
*   **Glow Effects**: Use `box-shadow` and `drop-shadow` sparingly to indicate "active" energy states.
*   **Micro-animations**: Use smooth, floating transitions (`cubic-bezier`) instead of linear snaps.

## 5. Chibi Bot Visual Identity
Each bot must follow a consistent design language:
- White/Silver main body.
- Glowing blue eyes.
- Unique role-based accessories (Crown, Magnifying glass, Paintbrush, etc.).

## 6. The 3D Agent Office (New Command Center)
The new command center uses a "Room-per-Agent" concept to visualize localized expertise and current task load.

| Component | Design Requirement |
| :--- | :--- |
| **The Room** | CSS 3D Cube or Isometric room. Semi-transparent glass walls to show internal "goals". |
| **Room Name** | Project-based neon signage above each room. |
| **Task Box** | A glowing, pulsing cube carried on the bot's head when `status: working`. |
| **Movement** | Hovering or walking animations within the room. |
| **Decoration** | Minimalist 3D props relevant to the role (e.g., a tiny server rack for Coder). |

## 9. Ultra-HD Elite Command (v5.0)
The Elite Command center prioritizes **HUD Clarity** and **Visual Comfort**. It moves beyond prototypes into a professional fleet management tool.

| Component | Elite Standard |

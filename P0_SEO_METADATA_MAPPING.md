### **Internal Linking Matrix (P2.1)**
- **Rule**: Every page MUST contain ≥3 internal links to related content.
- **Mapping**:
  - `Blog Post` -> Linked to: 1x Core Product, 1x Application Page, 1x FAQ/Contact.
  - `Product Page` -> Linked to: 1x Relevant Blog Post, 1x Application Page, 1x Resource (TDS/ROI).
  - `Application Page` -> Linked to: 2x Recommended Products, 1x FAQ.
